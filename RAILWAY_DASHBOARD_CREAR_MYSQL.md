# 🚂 RAILWAY - IR AL DASHBOARD Y CREAR MYSQL

## 📍 ESTÁS EN LA PÁGINA PRINCIPAL

---

## 🎯 PASO 1: IR AL DASHBOARD

1. **En la parte superior derecha**, busca el link **"Dashboard"**
2. **Click en:** "Dashboard"
3. **O ve directamente a:** https://railway.app/dashboard

---

## 🚂 PASO 2: CREAR PROYECTO NUEVO

1. **En el Dashboard**, verás tus proyectos (probablemente esté vacío)
2. **Click en:** "New Project" o "+ New Project" (botón grande)
3. **Selecciona:** "Empty Project" o "Blank Project"
   - **NO selecciones** "Deploy from GitHub" (no lo necesitamos)

---

## 🗄️ PASO 3: AGREGAR MYSQL DATABASE

1. **En tu proyecto vacío**, verás opciones como:
   - "+ New" o "+ Add Service"
   - O un botón para agregar servicios

2. **Click en:** "+ New" o "+ Add Service"

3. **Selecciona:** "Database" o busca en el marketplace

4. **Click en:** "MySQL" (debería aparecer como opción)

5. **Railway creará automáticamente** la base de datos MySQL

---

## ⏳ PASO 4: ESPERAR CREACIÓN

1. **Espera 1-2 minutos** mientras Railway crea MySQL
2. **Verás el progreso** en la pantalla
3. **Cuando termine**, aparecerá tu servicio MySQL en el proyecto

---

## 📋 PASO 5: OBTENER CREDENCIALES

1. **Click en tu servicio MySQL** (aparecerá en el proyecto)
2. **Click en la pestaña:** "Variables" o "Connect"
3. **Railway mostrará las variables:**
   - `MYSQLHOST` (o `MYSQL_HOST`)
   - `MYSQLUSER` (o `MYSQL_USER`)
   - `MYSQLPASSWORD` (o `MYSQL_PASSWORD`)
   - `MYSQLDATABASE` (o `MYSQL_DATABASE`)
   - `MYSQLPORT` (o `MYSQL_PORT`)

4. **COPIA estas credenciales** (las necesitarás para Render)

---

## ✅ RESUMEN DE ACCIONES

1. ✅ Click en **"Dashboard"** (arriba a la derecha)
2. ✅ Click en **"New Project"**
3. ✅ Seleccionar **"Empty Project"**
4. ✅ Click en **"+ New"** o **"+ Add Service"**
5. ✅ Seleccionar **"MySQL"**
6. ✅ Esperar creación
7. ✅ Copiar credenciales
